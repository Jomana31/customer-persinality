
# Customer Personality Analysis - Machine Learning Assignment

## Project Description
This project applies supervised and unsupervised machine learning algorithms to the Customer Personality Analysis dataset.

## Dataset
Dataset: Customer Personality Analysis  
File used: marketing_campaign.csv

## Tasks Completed
- Data loading and inspection
- Missing value handling using median imputation
- Feature engineering:
  - Age
  - TotalSpending
  - TotalChildren
- Data filtering
- Encoding categorical variables
- Regression models:
  - Linear Regression
  - Ridge Regression
  - Decision Tree Regressor
- Classification models:
  - Logistic Regression
  - KNN
  - Random Forest
- Clustering:
  - K-Means
  - Elbow Method
  - PCA visualization
  - Cluster profiling

## How to Run
1. Download the dataset from Kaggle.
2. Place marketing_campaign.csv inside a folder named data.
3. Open the Jupyter Notebook.
4. Run all cells from top to bottom.
